Subroutine to Response.write templates directly from arguments.

Parameters

(string) Template with placeholders.
(string[]) Array with the replacements.

Example 

' The infamous hello world example using Template.write
dim XString : set XString = new Template
XString.write "{0} {1}", array("Hello", "World")
XString.write "{0} {1}", null
set XString = nothing

test

public function test( sPattern,
bIgnoreCase,
bGlobal,
saArgs )

Compute the template and search for a match between the value and a relular expression.

Parameters

(string) XString representation of the regular expression
(boolean) true|false indicating to enable a CI search or not
(boolean) true|false indicating to match or not all occurrences of the pattern
(string[]) Replacements

Returns

true, if a match for the regular expression is found in the value false, otherwise

Example 

dim XString : set XString = new Template : XString.value = "{0} {1}"
Response.write XString.test("Hello", false, true, array("Hello", "World"))
Response.write XString.test("hello", true, true, array("Hello", "World"))
set XString = nothing
