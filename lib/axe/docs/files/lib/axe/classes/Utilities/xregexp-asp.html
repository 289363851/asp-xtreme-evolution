XRegExp

XRegExp is an open source (MIT License) JavaScript library that provides augmented and extensible regular expressions.  You get new syntax, flags, and methods beyond what ES3 and VBScript 5.5 support natively.  XRegExp is also a regex utility belt with tools to make your grepping simpler and more powerful.

Notes

  • A raw version of http://xregexp.com/.
  • Take a look at http://xregexp.com/api/ for the API.

constructor

Creates an extended regular expression object for matching text with a pattern.  Differs from a native regular expression in that additional syntax and flags are supported.  The returned object is in fact a native RegExp and works with all native methods.

Parameters

(string) Regex pattern string, or an existing regex object to copy.
(string) Any combination of flags.

Flags

g global
i ignore case
m multiline anchors
y sticky (Firefox 3+)
n explicit capture
s dot matches all (aka singleline)
x free-spacing and line comments (aka extended)

Returns

(RegExp) Extended regular expression object.

Example 

dim ReDate, Match
set ReDate = XRegExp( join( array( _
    "(?<year>  [0-9]{4} ) -?  # year " _
  , "(?<month> [0-9]{2} ) -?  # month" _
  , "(?<day>   [0-9]{2} )     # day  " _
), vbLf ) , "x" )
 
set Match = XRegExp.exec("2012-02-22", ReDate)
Response.write( Match.year )' -> '2012'
Response.write( vbNewline )

exec

Executes a regex search in a specified string.  Returns a match array or null.  If the provided regex uses named capture, named backreference properties are included on the match array.  Optional pos and sticky arguments specify the search start position, and whether the match must start at the specified position only.  The lastIndex property of the provided regex is not used, but is updated for compatibility.

Parameters

(string) String to search.
(regexp) Regex to search with.
(number) Zero-based index at which to start the search.
(mixed) Whether the match must start at the specified position only.  The string ‘sticky’ is accepted as an alternative to true.

Returns

(mixed) Match array with named backreference properties, or null.

Example 

dim ReDate, Match
set ReDate = XRegExp( join( array( _
    "(?<year>  [0-9]{4} ) -?  # year " _
  , "(?<month> [0-9]{2} ) -?  # month" _
  , "(?<day>   [0-9]{2} )     # day  " _
), vbLf ) , "x" )
 
dim pos, result
pos = 3
set result = JSON.parse("[]")
while( not isNull( XRegExp.exec( S("<1><2><3><4>5<6>"), XRegExp("<(\d+)>"), pos, "sticky" ) ) )
    set Match = XRegExp.exec( S("<1><2><3><4>5<6>"), XRegExp("<(\d+)>"), pos, "sticky" )
    result.push( Oi(Match, 1) )
    pos = Match.index + S( Oi(Match, 0) ).length
wend
Response.write( JSON.stringify( result ) )' result -> ['2', '3', '4']
Response.write( vbNewline )
