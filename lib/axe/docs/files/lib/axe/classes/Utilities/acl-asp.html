Acl

Acl provides a flexible Role Based Access Control (RBAC) implementation for privileges management.  In general, an application may utilize this utility to control access to certain protected objects by other requesting objects.

The features provided by this class are widely discussed and studied by the National Institute of Standard and Technology (NIST).  For the papers and more info visit http://csrc.nist.gov/groups/SNS/rbac/.

(string) resource identifier
(string) privilege identifier

Returns

Example 

dim AC : set AC = new ACL
set AC.Media = new ACL_Media_MSSQL
AC.Media.connectionString = "Provider=SQLOLEDB;..."
 
call AC.load()
Response.write( AC.isAllowed("nagaozen", "fire-spells", "cast") )
 
set AC = nothing
